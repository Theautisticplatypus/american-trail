# The American Trail
A retro Oregon Trail–inspired political and economic survival game.  
Navigate modern challenges like tariffs, regulation, public sentiment, and supply chain chaos.

## Features
- Retro parchment UI
- 20 advanced political and economic events
- Dynamic map progression
- Mini-games
- Faction reaction system

## How to Play
1. Click **Start Game**.
2. Read each event that appears.
3. Choose a decision.
4. Survive until Election Day.

## Folder Structure
- `index.html` – Main game UI
- `style.css` – Retro tan-paper theme
- `game.js` – Core game logic
- `minigames.js` – Mini-game functions
- `img/` – UI images
- `.github/workflows/` – GitHub Pages deployment

## Deployment
Push to GitHub and enable Pages:
```
Settings → Pages → Deploy from branch `main`
```
