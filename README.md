The American Trail
A browser-based strategy game inspired by the classic Oregon Trail, but set in the modern political and economic landscape.
Navigate tariffs, regulations, supply chain chaos, public sentiment, and random political events while keeping your business alive until Election Day.
Features
⦁	Dynamic random events
⦁	Animated UI (fade effects, travel bar, faction portraits)
⦁	Sound effects
⦁	Mini-games
⦁	Faction reactions
⦁	Animated start screen
⦁	Modular design
